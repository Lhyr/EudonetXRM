Informations générales

L'extension "Formulaire avancé" permet de créer des formulaires et des pages d'atterrissage optimisés pour des affichages sur tablette et mobile et respectant le RGAA (Référentiel général d'amélioration de l'accessibilité).

La création de formulaires avancés se fait par un assistant en plusieurs parties :

  • Une partie "Création graphique" permet de construire le formulaire à partir de blocs de structure, de contenu et des zones de saisie.
  • Une partie "Paramètres" donne accès aux propriétés du formulaire comme sa langue, le message à la validation, les dates de publication et les messages avant/après publication ainsi que les droits de visualisation/modification du formulaire par les autres utilisateurs de la base Eudonet.
  • Une partie "Publication" réunit le bouton de publication du formulaire, le lien de partage public et le code à copier pour intégrer le formulaire à un site existant.

Remarque

Pour bénéficier de toutes les fonctionnalités de création de formulaire, nous vous invitons à utiliser les navigateurs Chrome, Firefox ou Safari.

Création Graphique

L'onglet "Création graphique" vous permet de définir le contenu de votre formulaire en y déposant des blocs de structure, de contenu et des zones de saisie.


Les Blocs à déposer sont disponibles dans le menu droit sous l'onglet suivant :

Les blocs de structure

Bloc "Canvas" Ce bloc est obligatoire sur le formulaire, il est l'élément de base de la page sur lequel seront déposés les autres éléments.
Bloc "1 Colonne" Ce bloc permet d'ajouter une colonne qui prend toute la largeur disponible là où le bloc est déposé.
Bloc "2 Colonnes" Ce bloc permet d'ajouter 2 colonnes prenant chacune 50% de la largeur disponible là où le bloc est déposé. Les 2 colonnes passent l'une en dessous de l'autre sur les tablettes en mode portrait et les téléphones portables ou lorsque la largeur de la fenêtre se réduit.
Bloc "2 Colonnes 30%/70%" Ce bloc permet d'ajouter 2 colonnes prenant respectivement 30% et 70% de la largeur disponible là où le bloc est déposé. Les 2 colonnes passent l'une en dessous de l'autre sur les tablettes en mode portrait et les téléphones portables ou lorsque la largeur de la fenêtre se réduit.
Bloc "2 Colonnes 70%/30%" Ce bloc permet d'ajouter 2 colonnes prenant respectivement 70% et 30% de la largeur disponible là où il est déposé. Les 2 colonnes passent l'une en dessous de l'autre sur les tablettes en mode portrait et les téléphones portables ou lorsque la largeur de la fenêtre se réduit.
Bloc "3 Colonnes" Ce bloc permet d'ajouter 3 colonnes prenant chacune 1/3 de la largeur disponible là où le bloc est déposé. Les 3 colonnes passent l'une en dessous de l'autre sur les tablettes en mode portrait et les téléphones portables ou lorsque la largeur de la fenêtre se réduit.

Remarque

Chacun des blocs de structure a ses propres dimensions, marges et couleur de fond.

Modifier les dimensions, marges et couleur de fond d'un bloc depuis le menu droit

En sélectionnant un élément de la partie graphique, le menu droit se placera automatiquement sur l'onglet "Pinceau" permettant ainsi d'accéder aux propriétés de l'élément:

  • Les propriétés liées à la couleur de fond et la couleur, la forme et l'épaisseur des bordures sont réunies dans le chapitre "Couleur et bordure".
  • Les propriétés liées aux dimensions, aux marges et aux marges intérieures (padding) sont réunies dans le chapitre "Dimension et marge".

Les blocs de contenu

Le bloc "Texte"

Ce bloc permet d'ajouter une zone de texte dans le formulaire. Une barre d'outils permet de mettre le texte en gras, italique, souligné ou barré, d'ajouter un lien ou d'insérer des champs de fusion :



Les propriétés du texte comme l'alignement, la couleur, la police et la taille sont modifiables dans les chapitres Alignement et Typographie du menu droit


Comme pour les autres éléments, il est également possible de modifier les dimensions et les marges du bloc Image en ouvrant les menus correspondants dans le menu droit (voir chapitre Modifier les dimensions, marges et couleur de fond d'un bloc depuis le menu droit).

Ajouter un lien

Pour ajouter un lien, dans un bloc texte, sélectionner le texte qui portera le lien et cliquer sur l'icône en forme de maillon dans la barre d'outils.


Ensuite, cliquez une fois en dehors de la zone de texte puis cliquer sur le texte qui vient d'être transformé en lien (il devrait être souligner maintenant), vous verrez alors apparaître dans le menu droit les propriétés du lien.


Attention Il est important de commencer votre Href (URL du lien) par "https://" ou "http://" pour que le lien s'ouvre correctement.
N'hésitez pas à vérifier vos liens après avoir publié votre formulaire et avant de le partager.

Astuce pour coller en texte brut

Pour ne pas reprendre la mise en forme source lors d'un copier/coller, le navigateur "Chrome" propose une fonction "Coller en texte brut" dans le menu qui apparaît au clic droit.

Texte de remplacement d'une image

Afin d'assurer le respect RGAA par le formulaire, il est important de définir le texte de remplacement de l'image, il sera utilisé lorsque l'image ne peut pas s'afficher ou si l'internaute utilise un lecteur d'écran dans le cas d'un handicap visuel par exemple. Ce texte se définit dans les "paramètres de l'élément" qui s'affichent dans le menu droit au clic sur le bloc Image :

Changer l'alignement d'une image

Si votre image ne fait pas toute la largeur disponible de la zone et que vous souhaitez la centrer ou l'aligner à droite, il faut sélectionner l'élément parent et dans le menu "Alignement" du menu droit, sélectionner l'alignement qui vous convient.

Le bloc "Image"

Ce bloc permet d'ajouter une image dans le formulaire, par défaut elle ne dépasse pas la largeur disponible dans la zone où elle a été déposée et sa largeur se réduit automatiquement pour s'adapter à la taille de l'écran qui affiche le formulaire.
Comme pour les autres éléments, il est également possible de modifier les dimensions et les marges du bloc Image en cliquant sur le pictogramme "Pinceau" dans le menu droit (voir chapitre Modifier les dimensions, marges et couleur de fond d'un bloc depuis le menu droit).

Les blocs de saisie

Le bloc "Formulaire"

Ce bloc permet d'ajouter un ensemble d'éléments dont une zone centrée de 800px de large par défaut ainsi qu'un bouton de validation en bas de cette zone. C'est dans ce bloc Formulaire que pourront être ajoutés les blocs de saisie.

Modifier la couleur de fond du bloc Formulaire

Si vous souhaitez modifier la couleur de fond du bloc formulaire, il vous faut sélectionner l'élément au centre du bloc qui inclus une zone libre et le bouton.

Pour sélectionner cet élément, vous pouvez cliquer sur l'un des éléments du bloc formulaire et remonter en utilisant la flèche vers le haut de la barre d'outils ci-dessous :

Ensuite dans le chapitre "Couleur et bordure" du menu droit, vous pouvez modifier la "Couleur de fond" du bloc.

Modifier la largeur du bloc Formulaire

Si vous souhaitez agrandir ou réduire la largeur prise par le bloc formulaire, il vous faut sélectionner l'élément au centre du bloc qui inclus une zone libre et le bouton.

Pour sélectionner cet élément, vous pouvez cliquer sur l'un des éléments du bloc formulaire et remonter en utilisant la flèche vers le haut de la barre d'outils ci-dessous :

Ensuite dans le chapitre "Dimension et marge" du menu droit, la "Largeur max" par défaut de 800 px est visible vous pouvez alors la changer par une nouvelle largeur maximale en pixel ou en %.

Bloc "Texte Court" Ce bloc permet d'ajouter une zone de saisie d'un texte court, il permet à l'internaute renseigner par exemple son nom, prénom ou un texte sur une ligne.
Bloc "Courriel" bloc permet d'ajouter une zone de saisie d'un Courriel, à la saisie par l'internaute le format de l'adresse email entrée est vérifié.
Bloc "Téléphone" Ce bloc permet d'ajouter une zone de saisie d'un téléphone, il n'y a pas de contrôle spécifique au format du téléphone.
Bloc "Case à cocher" Ce bloc permet d'ajouter une case à cocher dans le formulaire.
Pour accéder aux propriétés de ce bloc, il faut cliquer sur la case à cocher.
Bloc "Numérique" Ce bloc permet d'ajouter une zone de saisie d'un numérique dans le formulaire.
Bloc "Date" Ce bloc permet d'ajouter une zone de saisie d'une date dans le formulaire.
La propriété "Heure" permet de définir si l'heure est également à saisir par l'internaute.
Bloc "Texte long" Ce bloc permet d'ajouter une zone de saisie d'un texte long dans le formulaire.
La propriété "Nombre de lignes" permet de définir la hauteur de la zone de saisie sur le formulaire. La saisie est limitée à 4000 caractères.
Bloc "Liste déroulante" Ce bloc permet d'ajouter une zone de saisie d'une liste déroulante à choix unique.
Bloc "Choix multiple" Ce bloc permet d'ajouter une zone de saisie d'une liste déroulante à choix multiple.

Modifier le libellé et les propriétés d'un bloc de saisie

Tous les types blocs de saisie listés ci-dessous fonctionnent selon le même principe, lorsqu'un bloc est déposé, il est composé de 2 éléments : un libellé et une zone de saisie.
En cliquant sur le libellé, on peut le modifier.

En cliquant sur la zone de saisie, on accède à ses propriétés dans le menu droit

Les propriétés varient en fonction du type de bloc de saisie mais certaines sont communes :

  • Texte en filigrane : Texte d'aide apparaissant dans la zone de saisie pour indiquer à l'internaute ce qui est attendu dans la rubrique. Le texte en filigrane disparait dès que l'internaute commence à saisir dans la rubrique. Si aucun filigrane n'est défini ici alors le filigrane paramétré dans Eudonet pour la rubrique associée (voir ci-dessous) s'applique.
  • Rubrique associée : Rubrique Eudonet dans laquelle est enregistrée la donnée saisie par l'internaute à la validation du formulaire. Si le formulaire est utilisé pour compléter/mettre à jour une fiche existante alors la rubrique est préremplie à l'ouverture avec la donnée présente dans Eudonet, c'est le cas par exemple pour un formulaire envoyé par email à des destinataires présents dans la base Eudonet.
  • Rendre cette obligatoire : Cocher cette case rend la saisie de cette rubrique obligatoire pour la validation du formulaire.

Les propriétés disponibles peuvent varier en fonction du type de bloc :

  • Case à cocher : Il n'y a pas de filigrane mais on peut définir si la case est cochée par défaut.
  • Date : Une option permet de définir si l'heure est également à saisir par l'internaute.
  • Texte long : Une propriété permet de définir la hauteur de la zone de saisie en nombre de lignes.

Exemple de création d'un formulaire avancé en quelques étapes

Voici le rendu de cet exemple de formulaire avancé une fois publié :

    Cet exemple est composé des blocs suivants :
  • Un bloc Canvas
  • Un bloc Formulaire
  • Un bloc Image
  • Deux blocs Textes
  • Un bloc 2 Colonnes
  • Quatre blocs de saisie

    Voici la suite d'étapes vous permettant de le construire :
  1. Tout d'abord, je définis la couleur de fond du formulaire, pour cela je clique dans la zone blanche (le "body") et, dans le chapitre "Couleur et bordure" du menu droit, je saisis la couleur de fond "#d0d0d0"
  2. Ensuite, je dépose un bloc "Canvas" qui formera la base du formulaire
  3. Puis un bloc "Formulaire" afin d'avoir une zone moins large et centrée sur laquelle je vais déposer les autres éléments et un bouton de validation. Je modifie ensuite la couleur de fond du bloc Formulaire en blanc (#FFFFFF) via le menu droit (voir Modifier la couleur de fond du bloc Formulaire)
  4. Je dépose dans le bloc Formulaire, un bloc Image en utilisant l'image de 800px x 300px : https://i.picsum.photos/id/299/800/300.jpg?hmac=SrYDbbLZfmUdQ4iM9kzvPJok4TJTKJz_vSM575jHbSY et je renseigne le texte de remplacement dans le menu droit.
  5. Je dépose un bloc Texte pour le Titre de mon formulaire, je copie/colle le texte "Restez informé(e) grâce à notre newsletter" (voir Astuce pour coller en texte brut) et je le mets en gras, ensuite je définis dans le chapitre "Alignement" du menu droit un alignement du texte centré et dans le chapitre "Typographie" la police Arial et une taille de police de 28px.
  6. Je dépose un deuxième bloc Texte pour la description de mon formulaire, je copie/colle le texte "Abonnez-vous maintenant pour recevoir par e-mail les dernières actualités, mises à jour, offres et plus encore (ainsi que d'autres actualités, produits, événements et promotions)." et je définis dans le chapitre "Typographie" du menu droit la police Arial et une taille de police de 20px
  7. Sous les textes, je dépose un bloc 2 colonnes
  8. Je dépose un bloc de saisie "Texte court" dans la colonne de gauche pour demander à l'internaute son Prénom, je modifie le libellé et j'affecte la "Rubrique associée" (voir modifier le libellé et les propriétés d'un bloc de saisie)
  9. Je dépose un bloc de saisie "Texte court" dans la colonne de gauche sous le prénom pour demander à l'internaute son Nom, je modifie le libellé et j'affecte la "Rubrique associée" (voir modifier le libellé et les propriétés d'un bloc de saisie)
  10. Je dépose un bloc de saisie "Courriel" pour demande à l'internaute son Adresse email, je modifie le libellé et j'affecte la "Rubrique associée" (voir modifier le libellé et les propriétés d'un bloc de saisie)
  11. Je dépose un bloc de saisie "Téléphone" pour demande à l'internaute son Téléphone, je modifie le libellé et j'affecte la "Rubrique associée" (voir modifier le libellé et les propriétés d'un bloc de saisie)

    12. Pour Aller plus loin :
  12. J'augmente la marge sous le bouton pour décaler un peu plus le bouton du bas du formulaire, pour cela je sélectionne le bloc formulaire en cliquant dans la zone blanche sous le bouton "Valider" et dans le chapitre "Dimension et marge" du menu droit, je passe la marge intérieure basse de 20px à 50px.
  13. Je renomme le bouton en "Je m'inscris" ,pour cela je clique sur le bouton et je change le texte dans le menu droit
  14. J'augmente les marges du Bloc de texte contenant la description afin que le texte soit moins collé au bord sur les écrans mobiles, pour cela je sélectionne le bloc Texte et dans le chapitre "Dimension et marge" du menu droit, je passe les marges intérieures gauche et droite de 5px à 20px.


Paramètres

L'onglet "Paramètres" donne accès aux propriétés du formulaire, comme sa langue, le message à la validation, les dates de publication et les messages avant/après publication, les droits de visualisation/modification du formulaire par les autres utilisateurs de la base Eudonet ainsi qu'aux options de personnalisation.

Paramètres généraux

Langue du formulaire

Elle est définie par défaut sur la langue de l'utilisateur, elle peut être changée afin de modifier la langue dans laquelle les messages de contrôles du formulaire s'afficheront, comme par exemple ceux pour les champs obligatoires non renseignées ou ceux pour une saisie d'adresse email incorrecte.

Message de remerciement

A la validation du formulaire par l'internaute, il est possible d'afficher un message de remerciement ou de le rediriger vers une autre URL internet. Dans le cas du message, il est possible de le mettre en page et d'y insérer des champs de fusion reprenant des informations que l'internaute vient de saisir.

Planification

Date de début

Cette date permet de définir la date à laquelle le contenu du formulaire sera accessible et le message à afficher si un internaute y accède avant cette date.

Date de fin

Cette date permet de définir la date à laquelle le contenu du formulaire ne sera plus accessible et le message à afficher si un internaute y accède après cette date.

Remarque

Les dates de début et de fin et les messages associés ne sont pris en compte que si le formulaire est publié.

Partage sur les réseaux sociaux

Les informations de partage paramétrées dans l'onglet "Publication", comme l'image, le titre et la description, sont prises en compte même si le formulaire est partagé avant d'être accessible.

Sécurité

Ces droits permettent de définir quels utilisateur Eudonet ont accès au formulaire en visualisation et/ou en modification.

Remarque

Les droits d'accès ne sont pas repris à la duplication d'un formulaire.

Personnalisation

Cette option permet de définir la couleur par défaut des liens dans le formulaire.

Ces options permettent de définir la couleur d'accentuation et la couleur de la valeur des champs de saisie.
La couleur d'accentuation correspond à la couleur du bouton et des champs pendant leur saisie par l'utilisateur.

Ces options permettent de définir la taille et la police des champs de saisie.

Remarque

La personnalisation est visible sur le formulaire publié mais n'est pas prise en compte à l'étape "Création graphique"

Publication

L'onglet "Publication" réunit le bouton de publication du formulaire, le lien de partage public, les liens de partage sur les réseaux sociaux et le code à copier pour intégrer le formulaire à un site existant.

Publier et partager

Publier

En activant le switch, le formulaire passe au statut "Publié" et devient accessible selon la planification définit dans les paramètres.

Remarque

A la publication, le formulaire est enregistré et tous les champs de saisies sont contrôlés afin de vérifier qu'ils ont tous une rubrique associée.

Lien public

Ce lien est disponible après avoir publié le formulaire et permet de partager le formulaire de manière publique.

Réseaux sociaux

Les 3 boutons permettent de partager le formulaire sur Facebook, LinkedIn ou Twitter. L'image, le titre et la description qui apparaîtront à la publication sont personnalisables.

Enregistrer les dernières modifications avant de partager

Après avoir terminé vos dernières modifications sur les informations de partage, pensez à enregistrer vos modifications afin qu'elles soient bien prises en compte au moment de partager votre formulaire.

Format et dimensions de l'image

L'image doit être au format png, svg, jpeg ou gif et ne doit pas dépasser 5 Mo.

Afin d'optimiser l'affichage de votre publication il est conseillé d'utiliser:

  • Pour Facebook, une image de 1200x630 px et d'au minimum de 200x200 px
  • Pour LinkedIn, une image de 1200x627 px
  • Pour Twitter, une image d'au minimum 600x335 px

Taille maximum pour le titre et la description

  • Le titre est limité à 70 caractères
  • La description est limitée 155 caractères

Intégrer le formulaire

Formulaire entier

Dès le formulaire publié, le code pour intégrer le formulaire dans une page d'un site web est disponible dans ce menu. Pour intégrer le formulaire, il faut copier le code et l'insérer dans le code source de la page web à l'endroit où vous voulez voir apparaître le formulaire. Le formulaire intégré correspond au formulaire public, il y a donc un reCAPTCHA Google V2 qui s'ajoute automatiquement au-dessus du bouton de validation.
