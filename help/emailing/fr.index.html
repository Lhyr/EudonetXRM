Création d'un email de masse

Editeur de mail

Bonnes pratiques

  • La largeur maximum d'un email doit être comprise entre 600px et 800px. Cela permettra d'assurer un meilleur affichage de l'email sur les petits écrans et lors de la prévisualisation de l'email dans les messageries.
  • Eviter les éléments HTML complexes et préférez une structure en grille plutôt que des éléments avec des positions fixe ou relative qui risquent de ne pas se positionner correctement sur tous les clients de messagerie.
  • La plupart des messageries bloquent par défaut l'affichage des images. Il est donc nécessaire de prévoir des textes de remplacements pour les images ("Alt") et éviter d'utiliser une image de fond pour l'email.
  • Pour améliorer la délivrabilité et l'affichage de l'email, il est conseillé de ne pas avoir qu'une seule image contenant toutes les informations ni d'avoir que des images sans texte dans l'email. Le ratio idéal est d'avoir 60% de texte et 40% d'image.
  • Préférez l'utilisation de Police de caractères compatibles avec un maximum de messagerie comme Arial, Verdana, Georgia ou Times New Roman.
  • Evitez le contenu nécessitant du Flash ou du Javascript qui sont mal supportés par les messageries et optez pour des .gif animés à la place.
  • Utilisez des requêtes media (media-query) pour améliorer la visibilité et l'accessibilité sur les petits écrans et les mobiles, en réorganisant les images et les textes, en augmentant la taille du texte et des boutons pour faciliter leur lecture et le clic depuis un mobile. En construisant votre email à partir des blocs que nous vous proposons, vous bénéficiez automatiquement de ces requêtes média.
  • Tester votre email avant l'envoi de votre campagne :
    • Tester l'affichage de l'email sur différentes messageries / différents écrans / en version mobile en utilisant la fonction "Envoyer un email de Test"
    • Utiliser des services comme www.mail-tester.com pour obtenir une note pour votre email prenant en compte plusieurs paramètres comme la validité des liens présents, les informations d'envoi, la proportion images/textes.
    • Tester votre email, en vous sélectionnant comme unique destinataire de votre campagne d'email.
  • Tester encore une dernière fois votre email avant de l'envoyer.

Pour bénéficier de toutes les fonctionnalités de création d’email, nous vous invitons à utiliser les navigateurs Chrome, Firefox ou Safari.

Etape 1 : Choix du modèle

    Vous pouvez choisir un modèle parmi :

  • Vos modèles : cette liste contient les emails que vous avez créés et ceux que les autres utilisateurs vous ont partagés.
  • Modèles personnalisables : vous y trouvez une liste de modèles responsives que nous avons mis à votre disposition en espérant que vous y trouverez votre inspiration.
  • Courriel simple : choisissez cette option si vous souhaitez réaliser un courriel simple ou si vous souhaitez repartir d'une page blanche.

Remarque

Le modèle "Courriel simple" a une largeur de 680px et est aligné à gauche et le modèle "Courriel centré" a un corps d'une largeur de 680px centré.

Astuce

Pour démarrer la création d'un nouveau courriel centré, choisissez le "Courriel centré" depuis la liste des modèles personnalisables.

Le bloc "Texte"

Vous pouvez faire glisser un bloc "Texte" directement dans votre email afin de pouvoir ensuite le modifier.

Vous pourrez ainsi éditer directement son contenu en utilisant les options de mise en page qui apparaissent dans la barre en haut de votre email.

Remarque

Pour Modifier la hauteur du bloc Texte cliquez dans l'espace vide à l'extérieur du Texte mais dans l'intérieur du Bloc Texte le menu droit affiche alors les propriétés du bloc, ouvrez le chapitre Dimension puis modifier la propriété Hauteur

Remarque

Pour modifier les marges intérieurs du bloc Texte cliquez dans la zone de texte, le menu droit affiche alors les propriétés du bouton, dans le chapitre Dimension puis modifier la propriété Marge intérieur

Le bloc "Image"

Vous pouvez faire glisser un bloc "Image" directement dans votre email afin de pouvoir ensuite le modifier.

Vous pourrez double cliquer sur l'image par défaut pour ouvrir le menu d'édition et sélectionner votre image.

Astuce

Un texte de remplacement peut-être défini dans les paramètres de l'image (Double-clic sur l'image). Il est fortement conseillé de définir un texte de remplacement ("Alt" ) sur vos images, il s'agit d'une brève description des images de votre email qui s'affiche lorsque les images ne sont pas accessibles. Certaines messageries désactivent par défaut les images, le texte de remplacement donne ainsi une indication sur la valeur de l'image à votre destinataire. Ce texte de remplacement est également utile pour l'accessibilité de votre email pour les destinataires qui utilisent un logiciel de lecture d'écran. De plus, la présence de texte de remplacement fait partie des bonnes pratiques pour la délivrabilité de votre email.

Astuce

Vous pouvez utiliser le bouton ci-dessous pour visualiser le rendu de votre email lorsque les images ne sont pas accessibles

Astuce

Il est fortement conseillé d'utiliser des images ayant déjà les dimensions que vous souhaitez pour votre email. Vous pouvez forcer la largeur et la hauteur dans les paramètres de l'image (Double-clic sur l'image) mais certaines messageries ne tiennent pas compte de ces paramètres.

Le bloc "Bouton"

Le bloc "Bouton" vous permet d'ajouter un bouton d'action dans votre email. Vous pouvez faire glisser un bloc "Bouton" dans votre email.

Remarque

les liens sont définis à l'étape 3 - Contenus et liens.

Un bouton avec une mise en forme par défaut s'affiche.

Changer la couleur du bouton

Sélectionner le bouton en cliquant dans la zone blanche à l'intérieur du bouton. Le menu droit affiche alors les propriétés du bouton, dans le chapitre "Couleur et bordure", l'option "Couleur de fond" vous permet alors de changer la couleur de fond du bouton.

Bords arrondis

Cela vous permet de définir la courbure des angles du bouton. Cette propriété n'est pas compatible avec la messagerie Outlook.

Bordure

Cela vous permet de définir la largeur, le style et la couleur des bordures. (Remarque : pour enlever les bordures du bouton, choisir le Style "none").

Couleur du texte du bouton

Cliquez dans la zone de texte à l'intérieur du bouton, pour éditer son contenu : une barre d'outils de mise en page s'affiche en haut de l'email et vous permet de mettre en page le texte.

Largeur du bouton

Sélectionner le bouton en cliquant dans la zone blanche à l'intérieur du bouton, le menu droit affiche alors les propriétés du bouton. Ouvrez le chapitre Dimension puis modifier la propriété largeur

hauteur du bouton

Sélectionner le bouton en cliquant dans la zone blanche à l'intérieur du bouton, le menu droit affiche alors les propriétés du bouton. Ouvrez le chapitre Dimension puis modifier les propriétés marges Haute et Basse

hauteur du bloc Bouton

Cliquez dans l'espace vide à l'extérieur du bouton mais dans l'intérieur du Bloc Bouton, le menu droit affiche alors les propriétés du bouton. Ouvrez le chapitre Dimension puis modifier la propriété Hauteur

Changer les bordures du bouton

Sélectionner le bouton en cliquant dans la zone blanche à l'intérieur du bouton, le menu droit affiche alors les propriétés du bouton, dans le chapitre Couleur et bordure, les options suivantes vous permettent de modifier l'aspect des bordures du boutons.

Le bloc "Séparateur"

Le bloc "Séparateur" vous permet de diviser visuellement le contenu de votre email.

Changer la hauteur du séparateur

Sélectionnez le séparateur en cliquant dessus, le menu droit affiche alors les propriétés du séparateur. Dans le chapitre Dimension et marge, l'option Hauteur (par défaut à 5px) vous permet de modifier la hauteur du séparateur.

Changer la couleur du séparateur

Sélectionnez le séparateur en cliquant dessus, le menu droit affiche alors les propriétés du séparateur. Dans le chapitre Couleur et bordure, l'option Couleur de fond vous permet de changer la couleur de fond du bouton.

Changer les marges du séparateur

Sélectionner le séparateur en cliquant dessus, puis ouvrez le chapitre Dimension et marge. En remontant ensuite à l'élément HTML ayant une marge haute de 5px par défaut, vous pourrez modifier les marges haute et basse de cet élément HTML.

Le bloc "Article"

Le bloc Article permet d'ajouter à votre email un ensemble composé d'une image, d'un titre, d'une zone de texte et d'un bouton, chaque élément pouvant être modifié comme s'il s'agissait d'un bloc unitaire

Les blocs "Colonne"

Bloc "1 Colonne" Il vous permet de créer une zone libre dans votre email où vous pourrez définir une couleur de fond pour y déposer ensuite d'autres blocs (Image, Texte, Article...)
Bloc "2 Colonnes" Il vous permet d'avoir 2 zones sur la même ligne pour y déposer vos blocs.
Bloc "2 Colonnes 30% / 70%" Il vous permet d'avoir 2 zones sur la même ligne pour y déposer vos blocs avec une colonne de gauche plus petite que celle de droite.

Astuce

Pour les blocs "2 colonnes" si vous souhaitez que vos éléments soient alignés verticalement, définissez pour chaque élément une hauteur identique correspondant au plus grand des 2 éléments.

"Les Blocs de mise en page"

Bloc "Entête" Il contient un lien de visualisation.
Bloc "Corps" Il contient un bloc "Texte" à modifier.
Bloc "Pied de page" Il contient sur la partie gauche les coordonnées à personnaliser, sur la partie droite les informations de liens et de réseaux sociaux et en dessous, le lien de gestion des abonnements. Vos liens personnalisés vers les réseaux sociaux sont à renseigner à l'étape "Contenu et liens".

Astuce

Sélectionner la cellule centrale et changer le paramètre "Largeur" du menu "Dimensions et marges" pour changer la largeur du bloc intérieur qui est par défaut à "680px".

Astuce

Sélectionner la cellule de fond et changer le paramètre "Couleur de fond" du menu "Couleur et Bordure" pour changer la couleur de fond de ce bloc, cela permet d'avoir une couleur de fond global pour l'email et une couleur différente pour ce bloc uniquement.

Astuce

Dans le bloc "Pied de page" vous pouvez retirer un réseau social, en sélectionnant la cellule et en cliquant sur le bouton "Supprimer".

Vous pouvez également remplacer l'un des réseaux sociaux proposé en double cliquant sur l'image pour la remplacer et changer le nom du lien.

Remarque

Les propriétés CSS sur le body ( body{} ) ne sont pas prises en charge par l'éditeur d'email.

Remarque

Le CSS présent dans la source de votre email est passé en inline. Certaines boites de messagerie ne tiennent pas compte du code CSS définis dans les balises <style> de votre code , c'est pourquoi nous transformons votre code CSS en inline afin de rendre l'affichage de votre code plus précis dans les différentes boites de messagerie.

Remarque

Pour modifier la feuille de style CSS, vous pouvez ouvrez le code .CSS de votre email via le bouton "CSS" à l'étape "Contenus et liens"

Autres opérations

Envoyer un email de test

Cette option est disponible àl'étape "2 : Création graphique" et à l'étape "3 : Contenus et liens". Elle vous permet d'envoyer un email de test à votre adresse ou à l'adresse de votre choix afin de tester le rendu de votre email sur différentes messageries et différents supports.

Remarque

Les liens de visualisation et de désinscription/gestion des abonnements ne sont pas fonctionnels dans ces emails de tests.

Enregistrer le modèle

Cette option est disponible à l'étape "2 : Création graphique" et à l'étape "3 : Contenus et liens". Elle vous permet d'enregistrer en tant que modèle l'email que vous avez construit ou modifié.

Sélection de couleur sous Firefox

Sous Firefox, la fenêtre de sélection de couleur se place par défaut avec une opacité de 0%. Pour que la couleur s'affiche il faut déplacer le bouton glissoire de transparence vers la droite.

Etape 3 : Contenus et liens

    Cette étape vous permet d'utiliser l'ensemble des fonctionnalités de mise en page pour modifier le texte de votre email, insérer des champs de fusion et définir des liens de redirection.

Remarque

L'insertion de tableau ou d'image en dehors de l'étape de création graphique est déconseillée car elle ne permet plus d'assurer le caractère responsive de votre email.

Les liens

Avant de sélectionner l'une de ses options, sélectionner le texte sur lequel le lien sera placé.

Lien vers une annexe Cette fonction vous permet de charger une annexe présente sur votre ordinateur sur le serveur et de placer un lien de téléchargement dans votre email.
Lien vers une url Cette option vous permet de placer dans votre email, un lien vers une URL externe.
Lien de désinscription & gestion des abonnements Cette option permet de placer un lien qui permettra au destinataire de se désinscrire ou de gérer ses abonnements en fonction du mode de gestion des désinscriptions. Remarque : ce lien n'est pas fonctionnel dans les emails de test
Lien de visualisation Cette option permet de placer un lien permettant d'ouvrir l'email dans un navigateur. Remarque : ce lien n'est pas fonctionnel dans les emails de test.
Lien vers un formulaire Cette option permet de placer un lien vers un formulaire Eudonet qui a été créé au préalable. ( Ce lien n'est pas fonctionnel dans les emails de test. )
Bouton d'affichage du CSS Cette option permet de modifier la feuille de style.

Astuce

Les blocs contenant du texte (Blocs "Texte", "Article", "Entête"...) n'ont plus de police de caractères prédéfinie, la police par défaut de l'email est défini dans la feuille de style .css et peut être modifiée à l'étape "Contenu et liens" via le bouton d'affichage du CSS.


Remarque

Cette feuille de style sera incluse dans la balise <HEAD> de votre email.

Le tracking des liens

Lien

Lorsque vous placez un lien dans votre email, vous pouvez définir des options de tracking.

Options de tracking

  • Désactiver le lien de tracking : cette option permet de ne pas tracer les statistiques de clics sur ce lien.
  • Nom du lien : Nommer le lien pour améliorer son accessibilité et retrouver plus facilement les statistiques associées.
  • Champ du lien : l'option "Champ du lien" vous permet de définir le champ de la fiche Cible/Invité/Destinataire qui sera mise à jour lors du clic sur le lien par le destinataire de l'email. Deux types de champ sont disponibles :
    • Case à cocher : La case à cocher dès le premier clic sur le lien.
    • Numérique : la valeur du champ numérique est incrémentée de 1 à chaque clic.

Etape 4 : Envoi de l'email

    A cette étape, vous pouvez :

  • Donner une description à votre campagne d'emailing.
  • Compter le nombre destinataires.
  • Définir la date d'expiration et de purge des liens de tracking.
  • Définir le mode d'envoi Immédiat, Planifié ou Récurrent.
  • Choisir de dédoublonner l'envoi sur les adresses emails.

La campagne est envoyée !

Ça y est, votre campagne est envoyée ! Vous pouvez aller consulter les statistiques de votre campagne dans l'onglet "Campagnes Mails" et suivre les clics sur les liens de trackings et les réponses aux formulaires sur les fiches Destinataires/Cibles/Invités.
